APIs

The APIs design allows the integration of new data of different formats and data models.


Collections API
high-level collection information (metadata: data format, size, etc.)
Search API
searching for collection items.
Annotation API
annotating existing data using W3C Web Annotation standard (mainly for manual annotations).
Data Enrichment API
collection enrichment through automatic mechanisms or by human interaction.
